# 🚀 The Distraction-Free YouTube Portal

A clean, minimal, and focused web portal designed for students and creators who want to learn from YouTube without the noise of ads, comments, and endless recommendations.

## 🔗 Live Demo
Experience the focus here: **[View Live Project](https://ritesh-builds-dev.github.io/My-First-Website/)**

## 🌟 Key Features
* **Distraction-Free Interface:** Watch high-quality educational content without sidebars or toxic comments.
* **Categorized Learning:** 12+ curated categories including Coding, Space, English Mastery, and more.
* **Focus-First Design:** Built using pure HTML logic to ensure lightning-fast loading and a simple user experience.
* **Responsive Navigation:** Seamlessly switch between Home, About, and Contact pages.

## 🛠️ Tech Stack
* **HTML5:** Structured the entire layout and content flow.
* **Advanced Tables:** Utilized nested HTML table structures to create a clean, organized grid for video content.
* **YouTube Embed API:** Integrated official YouTube frames for a seamless video-watching experience.

## 📖 Learning Outcomes
During this project, I mastered the core fundamentals of HTML, specifically:
* **Complex Layouts:** Structuring multi-row grids using nested tables.
* **Spacing Control:** Managing visual balance with `cellpadding` and `cellspacing` attributes.
* **Multi-Page Architecture:** Creating and linking a consistent theme across different pages (Home, About Us, Contact Hub).
